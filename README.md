# MiLAS AOS Core  
認知のOSを扱うための最小構造

MiLAS AOS Core は、認知の安定性・安全性・回復可能性を  
「OSとしての内部世界」という観点から扱うための  
基底アーキテクチャをまとめたリポジトリです。

本リポジトリは、以下の三段階構造を中心に整理されています。

---

## 1. MiLAS（Mind-in-Life Adaptive System）  
認知の内部世界を OS として捉えるための基底モデル。  
L1 / L2 / L3 の三層で、認知の動作・乱れ・回復を扱います。

- **L1：基底帰還**（反応・割り込み・自動化）  
- **L2：情報流動**（意味生成・再解釈）  
- **L3：境界透過**（方針・行動・安定化）

MiLAS は「認知がどのように動き、どのように乱れ、どのように戻るか」を  
最小限の構造で記述するための OS 的フレームです。

---

## 2. Recoverability（回復可能性）  
Zenodo / Preprints.org にて公開した仮説。  
認知の安定性を「壊れないこと」ではなく  
**“戻れること”** を基準に測定するための操作的概念。

外乱後にどの程度「元の認知状態へ復帰できるか」を扱います。

---

## 3. ISE（Inner Safety Engineering）  
MiLAS と Recoverability を応用し、  
認知の落下・境界破れ・解像度低下を扱うための  
**認知安全性の実務体系**。

- 認知の変調を検知する  
- どの層（L1/L2/L3）が危険かを見積もる  
- 回復可能性を確保する  
- 認知の安全設計を行う  

ISE は「認知の安全工学」として、  
MiLAS の上に構築される応用レイヤーです。

---

## Glossary（簡易版）  
詳細は `GLOSSARY.md` を参照。

- **Cognitive Drop**：認知の急落。解像度が一時的に低下する現象。  
- **Resolution Shift**：世界の見え方が粗くなる変調。  
- **Recoverability**：外乱後に「戻れる」能力。  
- **L1 / L2 / L3**：MiLAS の三層構造。  
- **ISE**：認知安全性の設計と実務。

## Structure（概念の流れ）  
詳細は `structure.md` にて段階的に整理予定。

- **L1 → ISE**：基底の安定性を観察する  
- **L2 → ISE**：情報流動の乱れを検知する  
- **L3 → ISE**：境界破れの兆候を扱う
